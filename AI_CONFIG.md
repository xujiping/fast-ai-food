# Fast AI Food - AI 助手配置指南

## AI 助手配置

Fast AI Food 应用支持两种方式配置 AI 助手：

### 方法一：在应用内配置（推荐）

1. 打开应用，点击首页的"设置"按钮
2. 在设置页面中，找到"AI 配置"部分
3. 关闭"演示模式"开关以启用 AI 助手
4. 输入您的 OpenAI API 密钥（格式：sk-...）
5. （可选）如果您使用的是自定义 API 端点，请输入"API 基础URL"
6. （可选）根据需要调整"文本模型"和"视觉模型"
7. 点击"测试连接"按钮验证配置是否正确
8. 点击"保存设置"按钮保存您的配置

### 方法二：使用环境变量

1. 复制 `.env.example` 文件为 `.env.local`
2. 在 `.env.local` 文件中设置您的环境变量
3. 重启应用

## OpenAI API 密钥获取

1. 访问 [OpenAI 官网](https://openai.com/)
2. 注册或登录您的账户
3. 进入 API 密钥页面
4. 创建新的 API 密钥
5. 复制密钥并粘贴到应用设置中

## 支持的模型

### 文本模型
- `gpt-3.5-turbo`（默认，性价比高）
- `gpt-4`（更强大的理解能力）
- `gpt-4-turbo`（更快的响应速度）

### 视觉模型
- `gpt-4o-mini`（默认，性价比高）
- `gpt-4-vision-preview`（更强大的图像理解能力）

## 常见问题

### Q: 为什么 AI 助手没有响应？
A: 请检查以下几点：
1. 确保已正确配置 API 密钥
2. 确保已关闭"演示模式"
3. 尝试点击"测试连接"按钮验证配置
4. 检查网络连接是否正常

### Q: 可以使用其他 AI 服务提供商吗？
A: 目前应用主要支持 OpenAI API，但您可以通过设置"API 基础URL"来使用兼容 OpenAI API 格式的其他服务。

### Q: API 密钥安全吗？
A: 您的 API 密钥将安全地存储在浏览器的本地存储中，不会上传到我们的服务器。但请注意不要与他人分享您的 API 密钥。

## 演示模式

如果您没有 API 密钥，可以保持"演示模式"开启。在演示模式下，AI 助手将提供预设的回复，让您体验应用的基本功能。

## 技术支持

如果您在配置过程中遇到问题，请：
1. 查看浏览器控制台的错误信息
2. 确认您的 API 密钥格式正确
3. 检查您的 OpenAI 账户余额是否充足

---

**注意**：使用 AI 助手可能会产生 API 调用费用，请根据您的使用情况合理配置。
